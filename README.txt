# Gammy - Personal Game Tracker

A beautiful, feature-rich game tracking application with a stunning Nebula dark theme.

## Features

- 🎮 **Personal Library** - Track your game collection
- ⭐ **Wishlist** - Save games you want to play
- 📔 **Game Diary** - Document your gaming sessions
- 📅 **Release Calendar** - Never miss a game release
- 📰 **Gaming News** - Stay updated with latest news
- 🌍 **Bilingual** - Full support for Italian and English
- 🎨 **Glassmorphism UI** - Beautiful modern design
- 📱 **Fully Responsive** - Works on all devices
